# Assets

此目錄存放專案靜態資源。

## 邊界
- 包含所有 **永久資源**（images, fonts, styles, data, docs）以及 **臨時開發資源**（tmp）
- 不包含業務邏輯或元件程式碼
- 永久資源會打包到 build；tmp 資源僅供開發或測試

## 範例子目錄
- images/     → 圖片、icon、logo
- fonts/      → 字型檔案
- styles/     → 全局 CSS / SCSS / theme
- data/       → JSON 配置或靜態資料
- docs/       → 靜態文檔
- tmp/        → 開發臨時檔案
